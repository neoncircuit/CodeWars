# Convert a Boolean to a String

## Instructions

Implement a function which convert the given boolean value into its string representation.

Note: Only valid inputs will be given.

## Given Code
```
export const booleanToString = (b:boolean):string => {
  // your code here
};
```

## My Solution
```
export const booleanToString = (b:boolean):string => {
  // your code here
  return b.toString();
};
```
